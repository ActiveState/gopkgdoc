About

GoDoc displays documentation for Go packages on Bitbucket, Github, Launchpad and Google Project Hosting.

The source code for GoDoc is available on Github.

Feedback

Send your ideas, feature requests and questions to info@godoc.org. Report bugs using the Github Issue Tracker. Join the discussion at the GoDoc group.

go.pkgdoc.org is now godoc.org

We moved to a shorter and easier to remember domain name.

Don't worry about your links to go.pkgdoc.org. The go.pkgdoc.org domain will redirect to godoc.org for the foreseeable future.

Bookmarklet

The GoDoc bookmarklet navigates from pages on Bitbucket, Github Launchpad and Google Project Hosting to the package documentation. To install the bookmarklet, click and drag the following link to your bookmark bar: GoDoc

Writing documentation for GoDoc

GoDoc produces documentation from Go source code. The guidelines for writing documentation for the godoc tool apply to GoDoc.
