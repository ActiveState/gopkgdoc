GoPkgDoc

GoPkgDoc displays documentation for Go packages on Bitbucket, Github, Launchpad and Google Project Hosting.

File an issue to report bugs or suggest improvements. The source for this site is hosted on Github. Pull requests are welcomed.

The GoPkgDoc bookmarklet navigates from pages on Bitbucket, Github Launchpad and Google Project Hosting to the package documentation. To install the bookmarklet, click and drag the following link to your bookmark bar: GoPkgDoc

GoPkgDoc keeps a list of the packages viewed on this site.
