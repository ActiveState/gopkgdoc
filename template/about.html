About

GoPkgDoc displays documentation for Go packages on Bitbucket, Github, Launchpad and Google Project Hosting.

GoPkgDoc is hosted on Google App Engine. The source code available on Github.

Feedback

Send your ideas, feature requests and questions to the author via Github or Google Profiles. Report bugs using the Github Issue Tracker.
